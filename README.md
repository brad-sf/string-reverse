# StringReverse - JavaScript

## Installation
`npm install string-reverse`

## Usage
Node.JS:
```javascript
const {strReverse} = require('string-reverse');
console.log( strReverse("Hello") ); // olleH
```

ES6:
```javascript
import {strReverse} from 'string-reverse';
console.log( strReverse("Hello") ); // olleH
```
